Security Bulletin

Microsoft Security Bulletin MS08-061 - Kritické
===============================================

Chyby zabezpečení jádra systému Windows umožňují zvýšení oprávnění (954211)
---------------------------------------------------------------------------

Publikováno: 14. října 2008

**Verze:** 1.0

### Obecné informace

#### Shrnutí

Tato aktualizace zabezpečení řeší jednu veřejně známou a dvě soukromou osobou oznámené chyby zabezpečení jádra systému Windows. Místní útočník, který by tyto chyby zabezpečení zneužil, by mohl získat úplnou kontrolu nad postiženým systémem. Uvedené chyby zabezpečení nemohou zneužít vzdálení ani anonymní uživatelé.

Tato aktualizace zabezpečení je označena vysokým stupněm závažnosti pro všechny podporované verze systému Microsoft Windows 2000, Windows XP, Windows Server 2003, Windows Vista a Windows Server 2008. Další informace získáte v podčásti **Software obsahující tuto chybu a software neobsahující tuto chybu** této části.

Aktualizace zabezpečení řeší uvedené chyby zabezpečení opravou ověřování vlastností okna předávaných během procesu vytváření nového okna, opravou způsobu zpracování systémových volání z více podprocesů a opravou ověřování parametrů předávaných jádru systému Windows z uživatelského režimu. Další informace o těchto chybách zabezpečení získáte v podčásti Nejčastější dotazy pro konkrétní položku chyby v části **Informace o chybách zabezpečení**.

**Doporučení:** Společnost Microsoft doporučuje, aby si zákazníci tuto aktualizaci aplikovali co možná nejdříve.

**Známé problémy.** [Článek 954211 znalostní báze Microsoft Knowledge Base](http://support.microsoft.com/kb/954211) popisuje aktuálně známé problémy, s nimiž se mohou zákazníci setkat v případě instalace této aktualizace zabezpečení. V článku jsou také uvedena doporučená řešení těchto problémů.

Následující software byl testován za účelem zjištění, kterých verzí nebo edic se chyba týká. U dalších verzí nebo edic buď již skončila časově omezená podpora, nebo nejsou touto chybou postiženy. Chyba zabezpečení jádra systému Windows při vytváření okna – CVE-2008-2250  
--------------------------------------------------------------------------
  
<span></span>
Chyba zabezpečení umožňující zvýšení úrovně oprávnění je způsobena tím, že jádro systému Windows nesprávně ověřuje vlastnosti okna předávané během procesu vytváření nového okna. Útočník, který úspěšně zneužije tuto chybu zabezpečení, může spustit libovolný kód v režimu jádra. Útočník by tak mohl instalovat programy, zobrazovat, měnit či odstraňovat data nebo vytvářet nové účty s úplnými uživatelskými právy.
  
Tuto chybu zabezpečení lze zobrazit jako standardní položku v seznamu známých chyb zabezpečení a ohrožení pod číslem [CVE-2008-2250](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2008-2250).
  
#### Skutečnosti snižující závažnost rizika chyby zabezpečení jádra systému Windows při vytváření okna – CVE-2008-2250
  
Snížení rizika se týká nastavení, běžné konfigurace nebo obecného osvědčeného postupu existujícího ve výchozím stavu, který by mohl snížit závažnost zneužití chyby zabezpečení. V dané situaci mohou být užitečné následující faktory snížení rizika:
  
-   Ke zneužití této chyby zabezpečení by útočník musel mít platná oprávnění pro přihlášení a musel by být schopný se přihlásit místně. Tuto chybu zabezpečení nemohou zneužít vzdálení ani anonymní uživatelé.
  
#### Možná zástupná řešení chyby zabezpečení jádra systému Windows při vytváření okna – CVE-2008-2250
  
Společnost Microsoft zatím nenalezla žádné zástupné řešení této chyby zabezpečení.
  
#### Nejčastější dotazy týkající se chyby zabezpečení jádra systému Windows při vytváření okna – CVE-2008-2250
  
**Jaký je rozsah této chyby zabezpečení?**    
Jedná se o chybu zabezpečení, která by mohla vést ke zvýšení úrovně oprávnění. Útočník, který by tuto chybu zabezpečení úspěšně zneužil, by mohl spustit libovolný kód a získat úplnou kontrolu nad postiženým systémem. Útočník by tak mohl instalovat programy, zobrazovat, měnit či odstraňovat data nebo vytvářet nové účty s úplnými uživatelskými právy.
  
**Co tuto chybu zabezpečení způsobuje?**    
Jádro systému Windows nesprávně zpracovává vstup předaný z nadřazeného do podřízeného okna při vytváření nového okna.
  
**Co je to jádro systému Windows?**    
Jádro systému Windows je jádrem operačního systému. Poskytuje služby na systémové úrovni, například správu zařízení a paměti, přiděluje čas zpracování procesoru jednotlivým procesům a řídí správu chyb.
  
**K čemu by mohl útočník tuto chybu zabezpečení zneužít?**
