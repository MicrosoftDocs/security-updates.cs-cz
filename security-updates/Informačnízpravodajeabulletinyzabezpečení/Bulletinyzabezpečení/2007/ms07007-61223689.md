Security Bulletin

Microsoft Security Bulletin MS07-007 - Důležité
===============================================

Chyba zabezpečení ve službě WIA umožňuje zvýšení úrovně oprávnění (927802)
--------------------------------------------------------------------------

Publikováno: 13. února 2007

#### Shrnutí

**Komu je tento dokument určen:** Zákazníkům, kteří používají systém Microsoft Windows.

**Dopad této chyby zabezpečení:** Zvýšení úrovně oprávnění

**Stupeň maximální závažnosti:** Vysoký

**Doporučení:** Zákazníci by měli tuto aktualizaci nainstalovat co možná nejdříve.

**Nahrazené aktualizace zabezpečení:** Žádný

**Upozornění:** Žádný

**Software obsahující tuto chybu:**

-   Microsoft Windows XP Service Pack 2 — [stáhnout aktualizaci](http://www.microsoft.com/downloads/details.aspx?familyid=ce695e0e-938c-4fc6-a9a2-0eb9fc3e5512)

**Software neobsahující tuto chybu:**

-   Microsoft Windows 2000 Service Pack 4
-   Microsoft Windows XP Professional x64 Edition
-   Microsoft Windows Server 2003 a Microsoft Windows Server 2003 Service Pack 1
-   Microsoft Windows Server 2003 pro systémy s procesorem Itanium a Microsoft Windows Server 2003 Service Pack 1 pro systémy s procesorem Itanium
-   Microsoft Windows Server 2003 x64 Edition
-   Windows Vista

Uvedený software byl testován, zda neobsahuje tuto chybu. Pro ostatní verze buď již nejsou poskytovány opravy zabezpečení, nebo tuto chybu neobsahují. Pokud chcete zjistit dobu, po kterou je pro určitý produkt a verzi poskytována technická podpora, navštivte web [Zásady poskytování technické podpory pro produkty společnosti Microsoft](http://go.microsoft.com/fwlink/?linkid=21742).

Podrobné informace o chybě zabezpečení
--------------------------------------

<span></span>
#### Chyba zabezpečení služby WIA – CVE-2007-0210:

Byla zjištěna chyba zabezpečení umožňující [zvýšení úrovně oprávnění](http://go.microsoft.com/fwlink/?linkid=21142) v systému Windows XP Service Pack 2 ve způsobu, jakým služba WIA spouští aplikace. Tato chyba zabezpečení by mohla přihlášenému uživateli umožnit získat úplnou kontrolu nad systémem.

#### Skutečnosti snižující závažnost rizika spojeného s chybou zabezpečení služby WIA – CVE-2007-0210:

-   Tuto chybu může zneužít pouze útočník s platnými pověřeními pro přihlášení, která by mu umožňovala se místně přihlásit. Tuto chybu zabezpečení nemohou zneužít vzdálení ani anonymní uživatelé.

#### Možná zástupná řešení chyby zabezpečení služby WIA – CVE-2007-0210:

Společnost Microsoft testovala následující zástupná řešení. Přestože tato zástupná řešení neopravují samotnou chybu zabezpečení, pomáhají blokovat známé způsoby útoku. Případná omezení funkčnosti způsobená daným zástupným řešením jsou vedena v následující části.

-   **Zakázání služby WIA**

    Zakázáním služby WIA přispějete ke zvýšení ochrany postiženého systému před pokusy o zneužití této chyby zabezpečení. Chcete-li službu WIA zakázat, postupujte takto:

    1.  Klepněte na tlačítko **Start** a potom na příkaz **Ovládací panely** .Můžete také přejít na příkaz **Nastavení** a pak klepnout na příkaz **Ovládací panely**.
    2.  Poklepejte na ikonu **Nástroje pro správu**.
    3.  Poklepejte na položku **Služby**.
    4.  Poklepejte na položku **Windows Image Acquisition (WIA)**.
    5.  V seznamu **Typ spouštění** klepněte na položku **Zakázáno**.
    6.  Klepněte na tlačítko **Zastavit** a pak klepněte na tlačítko **OK**.

    Službu WIA je také možné zastavit a zakázat zadáním následujícího příkazu na příkazovém řádku:

    `sc stop stisvc & sc config stisvc start= disabled`

    **Dopad tohoto zástupného řešení:** Pokud službu WIA zakážete, nemusí být možné se připojovat nebo komunikovat s různými obrazovými zařízeními včetně digitálních kamer a skenerů. Proto doporučujeme toto zástupné řešení využít pouze v systémech, které nevyžadují komunikaci s digitálními obrazovými zařízeními.

#### Nejčastější dotazy týkající se chyby zabezpečení služby WIA – CVE-2007-0210:

**Jaký je rozsah této chyby zabezpečení? **
Jedná se o chybu zabezpečení, která by mohla vést ke [zvýšení úrovně oprávnění](http://go.microsoft.com/fwlink/?linkid=21142). Útočník, který by tuto chybu zabezpečení zneužil, by mohl získat úplnou kontrolu nad postiženým systémem. Útočník by tak mohl instalovat programy, zobrazovat, měnit či odstraňovat data nebo vytvářet nové účty s úplnými uživatelskými právy. Aby se mohl útočník pokusit tuto chybu zneužít, musel by být schopen místně se přihlásit k systému a spustit program.

**Co tuto chybu zabezpečení způsobuje?**
Nekontrolovaná vyrovnávací paměť ve službě WIA.

**Co je služba WIA?**
Služba WIA umožňuje programům pro zpracování obrázků, jako jsou například Microsoft Picture It! 2000, Kodak Imaging nebo Adobe Photoshop komunikaci s obrazovými zařízeními, jako jsou digitální fotoaparáty a skenery. Podporuje digitální fotoaparáty a skenery s nízkým i vysokým výkonem, umožňuje též načítání obrázků z fotoaparátů pro digitální video (DV) založených na standardu [IEEE 1394](http://www.1394ta.org/technology/) a webových kamer využívajících sběrnici Universal Serial Bus (USB).

**K čemu by mohl útočník tuto chybu zabezpečení zneužít?**
Útočník, který by tuto chybu zabezpečení zneužil, může získat úplnou kontrolu nad ohroženým systémem.

**Jak by mohl útočník tuto chybu zabezpečení zneužít?**
Aby mohl útočník tuto chybu zneužít, musel by být nejdříve schopen přihlásit se k systému. Útočník by mohl následně spustit speciálně vytvořený program, pomocí kterého by mohl zneužít tuto chybu zabezpečení, a získat tak úplnou kontrolu nad postiženým systémem.

**Které systémy jsou touto chybou ohroženy nejvíce?**
Touto chybou zabezpečení jsou ovlivněny systémy Windows XP Service Pack 2.

**Bylo by možné tuto chybu zabezpečení zneužít prostřednictvím Internetu?**
Ne. Útočník se musí místně přihlásit ke konkrétnímu systému, aby jej mohl napadnout.

**Co je podstatou této aktualizace?**
Tato aktualizace odstraní tuto chybu zabezpečení změnou způsobu, jakým služba WIA ověřuje délku zpráv před jejich odesláním do přidělené vyrovnávací paměti.

**Byla v době zveřejnění tohoto bulletinu tato chyba zabezpečení veřejně známá?**
Ne. Společnost Microsoft byla o této chybě informována důvěryhodným zdrojem.

**Měla v době zveřejnění tohoto bulletinu společnost Microsoft nějaké informace o tom, že tato chyba zabezpečení byla zneužita?**
Ne. Společnost Microsoft neměla žádné informace, které by naznačovaly, že daná chyba zabezpečení byla v době původního zveřejnění tohoto bulletinu zneužita k útoku na zákazníky, a nezaznamenala ani žádné důkazy nebo publikované příklady kódu, které by naznačovaly, že tato chyba byla zneužita.
